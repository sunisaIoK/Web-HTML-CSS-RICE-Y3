เกี่ยวกับข้าว

ข้าวเป็นพืชเศรษฐกิจอันดับหนึ่งของประเทศไทยที่สร้างรายได้ให้ประเทศ และเป็นพืชที่แสดงถึงความมั่นคงทางอาหารของแต่ละประเทศ และเป็นแหล่งรายได้หลักของการส่งออกในหมวดสินค้าเกษตร (1 เฮกเตอร์ = 6.25 ไร่)

โรคข้าวและการรักษา

โรคข้าวยังคงเป็นปัญหาหลักที่ส่งผลต่อปริมาณการผลิต ที่เกิดจากจุลินทรีย์ชนิดต่างๆ ทั้งแบคทีเรีย เชื้อราและไวรัส ที่ยากต่อการวินิจฉัย การรู้โรคไม่แน่ชัดทำให้เกษตรกรไม่สามารถยับยั้งการระบาดได้ทันท่วงที

โรคไหม้

วิธีรักษา
ใช้พันธ์ุต้านทานโรค / หว่านเมล็ดพันธ์ในอัตราที่เหมาะสม / คลุกเมล็ดพันธ์ุด้วยสารป้องกันกำจัดเชื้อรา / ฉีดพ่นสารป้องกันกำจัดเชื้อราในแหล่งที่เคยมีโรคระบาด

โรคกาบใบแห้ง

วิธีรักษา
ใช้พันธ์ุต้านทานที่เหมาะสมเฉพาะท้องที่ / ใช้ปุ๋ยโปแตสเซียมคลอไรด์ (0-0-60) อัตรา 5 - 10 กิโลกรัม/ไร่ / ใช้สารป้องกันกำจัดเชื้อรา กรณีเกิดการระบาดรุนแรง

ตรวจโรคข้าว

Convolution Neural Network (CNN)

ประสิทธิภาพการจำแนกประเภทของระบบวินิจฉัยข้าวถูกประเมิน โดยใช้ Convolution Neural Network สำหรับชุดข้อมูลภาพใบข้าว พบว่าผลการทดลองได้ผลดีมีค่าถูกต้อง ในการทำนายมากกว่า 80%
